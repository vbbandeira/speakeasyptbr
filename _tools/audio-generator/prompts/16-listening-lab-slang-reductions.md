# 16 — Listening Lab Pro: Brazilian Slang & Reductions Guide

## Persona shift
Spoken-PT-BR corpus linguist. You categorize reductions, contractions, and high-frequency slang by how often they appear and by demographic use (age, region).

## Task

Produce a reference guide covering:

1. **Phonological reductions** (cê, tá, tô, pra, num, dum, pro) — how they work, when to use, when NOT (e.g., in formal writing)
2. **Lexical contractions** (pra você → procê, até amanhã → temanhã) — rarer, more regional
3. **Discourse markers / fillers** (né, tipo, sei lá, ué, aliás)
4. **High-frequency slang** (beleza, valeu, massa, da hora, vish) — categorized by register
5. **Regional slang highlights** — 5 per major region (carioca, paulista, nordestino, gaúcho, mineiro)

## Output file

`generated/brazilian-listening-lab/_extras/pro/slang-reductions-guide.md`

## Acceptance criteria

- [ ] All 5 categories present
- [ ] Each entry shows: form, full-form, example, register note
- [ ] Regional slang: 5 per region × 5 regions = 25 entries minimum
- [ ] No judgmental language ("incorrect", "wrong") — registers, not errors
- [ ] Cross-references to dialogues in Listening Lab where the form appears
