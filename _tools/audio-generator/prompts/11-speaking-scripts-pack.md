# 11 — Speaking Kit Pro Extra: 200 Speaking Scripts Pack

## Persona shift
Script writer for actor training meets language teacher. You write short scripts (2–4 lines) that learners can shadow. Each has variations by register.

## Task

Produce 200 mini-scripts, organized by scenario:

- 40 — Restaurante / bar / café
- 30 — Transporte (Uber, ônibus, aeroporto)
- 30 — Compras (supermercado, feira, roupa)
- 30 — Social (conhecendo pessoas, festas, primeiro encontro)
- 30 — Trabalho (entrevista, reunião, email spoken)
- 20 — Emergências e saúde
- 20 — Situações específicas (hotel, banco, consulado)

Each script:
- 2–4 lines of PT-BR
- Provided in **3 registers**: formal, informal, regional (nordestino OR carioca)
- No English translation inline (that's for a separate transcripts doc if needed)

## Output file

`generated/speaking-pronunciation-kit/_extras/pro/200-speaking-scripts.md`

## Output schema

```markdown
# 200 Mini Speaking Scripts — by scenario

---

## 🍽 Restaurante / Bar / Café (40 scripts)

### Script 001 — {situation, e.g., "Ordering coffee at a padaria"}

**Formal:**
- {A:} Bom dia, por favor.
- {B:} Bom dia, pois não.
- {A:} Um café expresso, por gentileza.

**Informal:**
- {A:} E aí, bom dia!
- {B:} Bom dia!
- {A:} Um café, por favor.

**Regional (carioca):**
- {A:} E aí, brother, bom dia!
- {B:} Bom dia, chefe.
- {A:} Um café aí, por favor.

---

### Script 002 — ...

...

## 🚗 Transporte (30 scripts)

...
```

## Acceptance criteria

- [ ] Exactly 200 scripts
- [ ] Distribution matches the breakdown above
- [ ] Each script has 3 registers
- [ ] No script exceeds 5 lines total
- [ ] Regional variant alternates between nordestino and carioca across the 200 (roughly half and half)

## Armadilhas comuns

- Don't make formal scripts stilted. Formal in BR-PT is still warm.
- Don't make informal sound illiterate. "Vc", "pq", etc. are writing reductions, not speech.
- Don't make regional caricatures. Subtle markers, not heavy stereotypes.
